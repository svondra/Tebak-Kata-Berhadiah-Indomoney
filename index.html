Cara main

Tebak kata tengah dari kode kupon rahasia IndoMoney.

  1. Lihat pola kode kupon dan baca hint.
  2. Ketik kata tengah, lalu tekan Enter.
  3. Hijau berarti karakter benar dan posisinya tepat.
  4. Kuning berarti karakter ada di kode, tetapi posisinya belum tepat.
  5. Abu-abu berarti karakter tidak ada di kode.
I Benar dan tepat
N Ada huruf dalam kata, tapi posisi beda
X Tidak ada
